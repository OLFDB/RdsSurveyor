Driver for OLFsDABRadio based on ugreen DAB board in FM mode
============================================================

This board exposes RDS data via BLE.

You need to download the simplejavable-vx.xx.x.jar from https://github.com/simpleble/simpleble/releases.
As of SimpleBLE v0.12.1 macos x64 is not supported, while macos and Linux aarch64 is supported as well as 
Windows and Linux x64. You need to build it on your own from the sources for macos x64.

Keep in mind that SimpleBLE is NOT free software. See https://github.com/simpleble/simpleble/blob/main/LICENSE.md for details.

Copy the simplejavable-vx.xx.x.jar into the lib dir under RDSSurveyor.
